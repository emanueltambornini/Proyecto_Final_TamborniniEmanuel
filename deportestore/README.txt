# La Aplicacion permite la Funcionalidad de un Comercio de Venta de Ropa Deportiva y tambien
muestra las Sucursales de Tiendas donde los Clientes pueden Comprar de Forma Presencial

Para Ingresar a la Aplicacion:
Usuario: admin
Clave: Ema1984!

No Pude hacer andar el "Logout", no pude encontrar el error.


- Los Modelos que Presenta esta Aplicacion son:

 -Productos
 -Medios de Pago
 -Sucursales

- Orden de Prueba de la Aplicacion:

1# Para Crear los Productos disponibles para la Venta en la Pagina usamos:
http://localhost:8000/aplicacion/productos_form/

Esta Tabla contiene dos campos el Nombre del Producto y el Precio

2# Para Crear los medios de pago disponibles para los clientes usamos:
http://localhost:8000/aplicacion/medios_pago_form/

Esta Tabla contiene solo un Campo que es el Nombre del medio de pago

3# Para Crear las Sucursales de los Locales de Venta Presencial usamos:
http://localhost:8000/aplicacion/sucursales_form/

Esta Tabla contiene dos campos el Nombre de la Ciudad y la Direccion

4# Para Ir a la Pagina de Inicio usamos:
http://localhost:8000/aplicacion/

Ac√° vamos a encontrar la Pagina Principal con el Nombre de la Tienda "deportestore"
y los accesos al Menu de Productos - Medios de Pago - Sucursales

5# Para consultar los Productos disponibles en el sitio usamos:
http://localhost:8000/aplicacion/productos/

6# Para consultar los medios de pago del sitio usamos:
http://localhost:8000/aplicacion/medios_pago/

7# Para consultar las Sucursales para compra presencial usamos:
http://localhost:8000/aplicacion/sucursales/

8# Para la busqueda de un Producto usamos un Patron de busqueda en Productos:
http://localhost:8000/aplicacion/buscar/
